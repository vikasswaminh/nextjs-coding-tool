# nextjs-coding-tool

A small Next.js-based coding/tooling project. This repository is intended for experimenting with Next.js features, building utilities, and iterating quickly on ideas.

## Getting Started

### Prerequisites

- Node.js (LTS recommended)
- npm / pnpm / yarn

### Install

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Then open http://localhost:3000

## Scripts

Common scripts you may use:

- `npm run dev` — start the development server
- `npm run build` — create a production build
- `npm run start` — start the production server
- `npm run lint` — run linting
